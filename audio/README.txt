Audio Files Setup:
==================

1. Place your audio files in this 'audio/' folder
2. Supported formats: MP3, WAV, OGG
3. Recommended: Short sound effects (1-5 seconds)

Example audio files referenced in script.js:
- panel1_sound.mp3
- panel3_sound.mp3

You can generate audio using:
- Online tools like freesound.org
- Audio editing software (Audacity, Adobe Audition)
- AI audio generators
- Text-to-speech for dialogue

Tips:
- Keep file sizes small for faster loading
- Use appropriate audio for the panel mood
- Test volume levels for consistency
- Consider background music vs sound effects
